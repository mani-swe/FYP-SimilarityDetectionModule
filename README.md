# FYP-SimilarityDetectionModule

This model is used to find similarity between 2 sentences and give back similarity percentage, my use for this module is to get a new FYP title and compare with previous store FYPs to check whether similar project is present in database and how much it is similar to other FYPs 

Download Vectors Library used in this program through these links:
Google Drive: https://drive.google.com/file/d/1GoQFoeqYduIXNB_XmlnHxeN9Urms9uX7/view?usp=sharing
Official Site: https://nlp.stanford.edu/data/wordvecs/glove.2024.wikigiga.50d.zip

Please Make Sure To Enter Correct Path Of Vectors File in Program
